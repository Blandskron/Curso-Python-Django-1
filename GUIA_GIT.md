# Guía de Trabajo: Curso Python Django

Has clonado este repositorio para trabajar localmente. Aquí tienes los comandos que necesitas para mantener tu flujo de trabajo.

## Tu Estado Actual
- **Rama principal (intacta):** `main` (esta sigue al repositorio original en GitHub)
- **Tu rama de trabajo:** `mi-trabajo` (aquí harás tus ejercicios y cambios)

Estas actualmente en la rama `mi-trabajo`.

## Cómo actualizar tu repositorio
Cuando el profesor (o el dueño del repo original) suba nuevo contenido:

1. Vuelve a la rama principal:
   ```bash
   git checkout main
   ```

2. Descarga los últimos cambios del internet:
   ```bash
   git pull origin main
   ```

3. Vuelve a tu rama de trabajo:
   ```bash
   git checkout mi-trabajo
   ```

4. Fusiona los nuevos cambios en tu trabajo:
   ```bash
   git merge main
   ```

¡Y listo! Tendrás lo nuevo sin haber perdido tus cambios.
